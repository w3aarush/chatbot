# File Structure

> **chatdatabase** - vector database folder
<p>This folder has all the chatdata inside it. Data can be accessed by creating a client to this database.</p>
<hr>

> **chatdata.csv** - csv file for the chatdata
<hr />

> **chatdata_backup.csv** - backup file for the chatdata
<hr />

> **db_creation.ipynb** - Jupyter notebook file used for creating the vector database using chroma
<hr />

> **db_preparaton.ipynb** - Jupyter notebook file used for preparing the csv file.
<hr />
